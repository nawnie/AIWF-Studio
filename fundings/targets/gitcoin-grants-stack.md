# Gitcoin Grants Stack

Official site: https://www.gitcoin.co/grants-stack

## Fit

Medium fit. Gitcoin can work if AIWF Studio is framed as public-goods local AI infrastructure and if there is a real community willing to make small contributions.

Best angle:

- Open-source local AI public good.
- Reduces dependence on centralized AI services.
- Helps users keep prompts, images, models, and training data local.
- Builds reusable worker/model infrastructure.

## What You Need

- Relevant active round.
- Wallet and Gitcoin profile.
- Public repo and project page.
- Short grant page copy.
- Community outreach plan.
- Proof of legitimacy: screenshots, tests, docs, demos.
- Matching eligibility requirements for the specific round.

## Grant Page Draft

AIWF Studio is open-source local AI infrastructure for people who want powerful creative AI workflows without relying on centralized cloud platforms. It supports image generation, inpainting, ControlNet-style workflows, Wan video experiments, model operations, local chat, and training services through a clean Python architecture.

Funding supports worker isolation, GPU tenant safety, local model operation tooling, documentation, and benchmarks so more people can run serious AI workflows on their own hardware.

## Round Strategy

1. Do not apply cold with no community.
2. Update GitHub README first.
3. Post a demo and explain the public-goods angle.
4. Ask small supporters for early contributions.
5. Keep the grant page very concrete: what exists, what funding unlocks, why local AI is a public good.

## Milestones

1. Public v2 release branch.
2. Worker and GPU tenant docs.
3. Local privacy/security guide.
4. Consumer-GPU benchmark logs.
5. Training/video/chat demo.

## Caveats

Gitcoin outcomes depend heavily on round eligibility, matching pools, and community participation. Treat this as a community-building target, not a guaranteed grant.
